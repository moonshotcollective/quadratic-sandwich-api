# quadratic-sandwich-api

```
endpoints: 
    - /api/projects/                GET All Projects
    - /api/projects/<ID>            GET Project specified by <ID>
    - /api/ballots                  GET All Projects
    - /api/op/projects/new          POST Upload new projects
    - /api/op/projects/rm/<ID>      DELETE Project specified by <ID>
    - /api/holder/ballots/cast      POST Upload cast ballot
```
